## Projecten

### 🔒 LD2410 Lock
*Vergrendel automatisch je pc als je weggaat*

Een Windows tray-applicatie die een **HLK-LD2410C mmWave radarsensor** gebruikt om aanwezigheid te detecteren. Zodra je weggaat van je bureau, vergrendelt de pc automatisch.

- Verbinding via **Bluetooth LE** of **UART**
- Instelbare gevoeligheid en vergrendelvertraging
- Radarsensor visualisatie
- MSI installer (perUser)

[→ Bekijk repository](https://github.com/gijsbertsonnevelt/LD2410Lock)

---

### 📥 yt-dlp Queue
*YouTube video's en playlists eenvoudig downloaden*

Een Windows downloadmanager gebouwd op **yt-dlp**. Zoek video's en playlists rechtstreeks vanuit de app, voeg ze toe aan de wachtrij en download ze als MP4 of MP3.

- Zoeken op YouTube (video's en playlists)
- Parallel downloaden met wachtrij
- MP4 (H.264) en MP3 ondersteuning
- Automatische update-controle voor yt-dlp, ffmpeg en deno
- MSI installer (perUser)

[→ Bekijk repository](https://github.com/gijsbertsonnevelt/YtDlpQueueDesktop)

### 🔒 LD2410 Lock

A Windows tray application that uses an **HLK-LD2410C mmWave radar sensor** to detect presence. When you leave your desk, the PC locks automatically.

### 📥 yt-dlp Queue
*Download YouTube videos and playlists with ease*

A Windows download manager built on **yt-dlp**. Search for videos and playlists directly from the app, add them to the queue and download as MP4 or MP3.

- YouTube search (videos and playlists)
- Parallel downloading with queue
- MP4 (H.264) and MP3 support
- Automatic update checks for yt-dlp, ffmpeg and deno
- MSI installer (perUser)

[→ View repository](https://github.com/gijsbertsonnevelt/YtDlpQueueDesktop)
